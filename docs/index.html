EntCord

EntCord란?

  • 엔트리에서 Discord Bot 을 작동시킬 수 있는 기능입니다.

  • 현재는 베타라서 오류가 많을 수 있습니다.

코드 참고: thoratica님의 EntBlocks을(를) 많이 참고하였습니다.

디스코드 봇 로그인 등: discord.js webpack (discord.js)

예제 작품

사용 방법

엔트리 작품 만들기에 들어가 개발자 도구(F12 or Ctrl + Shift + J) 콘솔에

$.get('https://raw.githack.com/ditto7890/EntCord/master/block.js') //최신 버전
$.get('https://raw.githack.com/ditto7890/EntCord/master/block_.js') //latest 버전

라고 입력해주시면 됩니다.
